This R script created a series of heat maps that characterized the usage distributions for V, D, J, and constant region genes.
Gene use distributions were calculated for two donors in four B cell compartments.
Data was gathered using BCR sequencing of two FACS sorted PBMC samples.
PBMC donors included one "BC Teen" who recovered from H5N1 infection in 2024 and one H5 naive "Golden Ticket Donor".
B cell compartments sequenced were characterized by reactivity to H5 hemagglutinin and classification as naive or memory cells.
